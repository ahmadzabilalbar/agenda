# 📅 Agenda 2026

A high-performance, single-file offline web application for academic and logistics planning.

## 🚀 Features
- **Zero Dependencies**: Pure HTML/CSS/JS in a single `index.html` file.
- **PWA-Lite**: Installable on iPhone/Android via "Add to Home Screen".
- **High Density**: Optimized for small screens with a badge-based layout.
- **Data Portability**: JSON-based Backup/Restore system.
- **WhatsApp Ready**: One-click export for sharing schedules via WhatsApp.
- **Theme Support**: 3-way toggle (Light, Dark, System Auto).

## 🏠 Installation
1. Download `index.html`.
2. Open in any browser.
3. **iPhone**: Share ➔ Add to Home Screen.
4. **Android**: Menu (⋮) ➔ Install App / Add to Home Screen.

## 🛠️ Tech Stack
- Vanilla HTML5
- Vanilla CSS3 (with Glassmorphism)
- Vanilla ES6+ JavaScript
- localStorage for persistence
